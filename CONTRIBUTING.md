## Roadmap and Status

The high-level ambitious plan for the project, in order:

|  #  | Step                                                      | Status |
| :-: | --------------------------------------------------------- | :----: |
|  1  | Working prototype for the memory model                    |   ✅   |
|  2  | Training a better standalone memory model                 |   ❌  |
|  3  | Make Tiles a proxy, with local model as memory module     |   ❌   |
|  4  | Public key cryptography with private key on device        |   ❌   |
|  5  | AT Proto integration                                      |   ❌   |
|  6  | Cross-platform `tilekit` SDK for developers               |   ❌   |
|  7  | Linux support with Flatpaks                               |   ❌   |
|  N  | Fancy features (to be expanded upon later)                |   ❌   |

Additional details for each step in the big roadmap is listed as issues in this repository.

Feel free to contribute to the project by submitting issues or pull requests aligned with the roadmap scopes. New inference backends and memory models are welcome.
